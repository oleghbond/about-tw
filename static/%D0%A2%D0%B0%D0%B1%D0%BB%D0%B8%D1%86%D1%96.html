Основи

Для форматування таблиць у TiddlyWiki5 використовується знак вертикальна риска | ось так:

|!Клітинка1 |!Клітинка2 |
|Клітинка3 |Клітинка4 |

Знаки оклику використовуються для позначення клітинок заголовків. Наприклад:

Клітинка1Клітинка2
Клітинка3Клітинка4

Вирівнювання в клітинках

Вирівнювання вмісту в клітинках таблиці контролюється шляхом вставки символів пробілу до і/або після вмісту клітин. Наприклад:

|Текст праворуч |
| Текст праворуч|
| Текст по центру |
|+++ дуже широка колонка, щоб бачити вирівнювання +++|

Що відображається у вигляді:

Текст праворуч
Текст праворуч
Текст по центру
+++ дуже широка колонка, щоб бачити вирівнювання +++

Вертикальне вирівнювання в клітинках

Вертикальне вирівнювання вмісту клітинок проводиться шляхом вставки в якості першого символу клітини

  • ^ для вирівнювання вгору,
  • , для вирівнювання вниз.

Нормальне горизонтальне вирівнювання як і раніше можливо. Наприклад:

|^вгору ліворуч |^ вгору центр |^ вгору праворуч|
|посередині ліворуч | посередині центр | посередині праворуч|
|,вниз ліворуч |, вниз центр |, вниз праворуч|

що відображається таким чином:

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
::
::		вгору ліворучвгору центрвгору праворуч::
::
::
::		посередині ліворучпосередині центрпосередині праворуч::
::
::
::		вниз ліворучвниз центрвниз праворуч::
::
::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::

Якщо вам потрібно мати ^або , як перший знак ліворуч-вирівняної клітинки, то ви можете використати HTML-код цих символів:

^&#94;
,&#44;

Злиття клітинок

Щоб об'єднати одну клітинку таблиці з іншою вище, використовуйте спеціальний символ ~. Щоб об'єднати одну клітинку з іншою ліворуч використовуйте символ <. Щоб об'єднати одну клітинку з іншою праворуч використовуйте знак >. Наприклад:

|Cell1 |Cell2 |Cell3 |Cell4 |
|Cell5 |Cell6 |Cell7 |<|
|Cell5 |~|Cell7 |Cell8 |
|>|Cell9 |Cell10 |Cell11 |

Що відображається у вигляді:

Cell1Cell2Cell3Cell4
Cell5Cell6Cell7
Cell5Cell7Cell8
Cell9Cell10Cell11

Класи, підписи, заголовки, нижні колонтитули

CSS класи, заголовки, нижні колонтитули таблиці можуть бути визначені як спеціальні псевдо-рядки. Наступним прикладом:

  • класи CSS "myClass" і "anotherClass" призначені таблиці;
  • таблиці надано підпис "Це підпис";
  • додано рядок заголовків з текстом "Заголовок" у кожній клітинці;
  • додано рядок нижнього колонтитула текстом "Колонтитул" у кожній клітинці.
|myclass anotherClass|k
|Це підпис |c
|Клітинка1 |Клітинка2 |
|Клітинка3 |Клітинка3 |
|Заголовок|Заголовок|h
|Колонтитул|Колонтитул|f

Що відображається у вигляді:

Це підпис
Клітинка1Клітинка2
Клітинка3Клітинка3
ЗаголовокЗаголовок
КолонтитулКолонтитул
