Створення документів

Установка

Цей плагін розроблений Jed Carty і його можна скачати і переглянути його в дії з авторського сайту.

Для установки плагіну зайдіть на авторському сайті на закладку Plugins на Control Panel. Знайдіть у списку встановлених плагінів A plugin to help create setup wizards or slideshows і перетягніть його до своєї Тідлівікі. Це вся установка.

Після встановлення серед іншого до вашої вікі буде скопійовані два файли:

  • $:/plugins/inmysocks/WizardWizard/Wizard Macro
  • $:/plugins/inmysocks/WizardWizard/Wizard Setup

Ці файли містять пояснення англійською мовою. У разі необхідності українізації цих текстів перетягніть ці два файли з цієї вікі до своєї.

Призначення

Найпростішим визначенням дії цього плагіну є "слайд-шоу". Ви лише задаєте тег і в результаті ви можете рухатися вперед і назад через список тиддлерів з цим тегом. Для управління гортання сторінками слайд-шоу вам надаються автонумерація слайдів, кнопки вперед і назад, а також низка посилань на слайди, що ви вже переглянули. Все - і налаштування слайд-шоу і управління - інтуїтивно просто.

Використання

Запустіть майстер налаштування презентації (слайд-шоу або краще тідлер-шоу), перейшовши до $:/plugins/inmysocks/WizardWizard/Wizard Setup. Там ви оберете тег, за яким буде відфільтрована колекція тідлерів для презентації, і встановити їхній порядок, а також деякі інші параметри.

Далі створіть тідлер, що власне стане контейнером презентації, надайте йому назву, а потім будь-де у тексті вставте <<ConfigurationWizard WizardName>>, замінивши WizardName на ім'я тегу, на базі якого утвориться список тідлер-слайдів презентації (ім'я цього тегу власне і буде назвою презентації). Voila, презентація готова.

Повна команда (макрос) запуску презентації має такий вигляд:

<<ConfigurationWizard WizardName SortField class>>
WizardNameНазва презентації (ім'я тегу колекції тідлерів). Відповідно, різні презентації мають різні назви.
SortFieldПоле тідлера для впорядкування тідлерів презентації. За замовчанням це поле order.
classДодатковий css клас, за детальнішою інформацією див. нижче.
UseAliasЯкщо встановлено значення true тоді тідлер, що містить презентацію, показує в якості заголовка поле caption поточного тідлер-слайда замість його назви (поле title). Якщо поточний тідлер-слайд не має поля caption, то тоді у якості заголовка використовується його назва (поле title). Значення за замовчанням false.

Майстер налаштування презентації також In the tiddlers for each step of the wizard you have access to some special helper macros. These macros only actually function in the wizard itself, so if you open up one of the tiddlers for a specific step these macros won't function as expected.

  • <<WizardTemp>> це тимчасовий тідлер для запам'ятовування введення на будь-якому слайді. Наприклад, ви забажаєте ввести текстове значення на одному зі слайдів для змінної name з допомогою такого кода <$edit-text tiddler=<<WizardTemp>> field=name/>. Тоді на одному з наступних слайдів ви зможете використати це введене вами значення поля name таким чином <$tiddler tiddler=<<WizardTemp>>>{{!!name}}</$tiddler>.
  • <<WizardState>> це тідлер стану для запам'ятовування інформації щодо поточного стану презентації. Ім'я тідлера поточного слайда тримається у полі current_step тідлера стану. Ви можете використовувати його для зберігання іншої інформації про стан або, якщо ви хочете керувати поточним кроком, ви можете змінити вміст поля. Більш детально ви можете розібратися переглянувши авторську презентацію.

Зробіть кнопку завершення

Коли ви знаходитесь на останньому слайді, кнопка Forward (Вперед) не з'явиться. Замість цього ви можете створити тідлер, позначений тим же тегом, що і всі інші у цій презентації. Потім встановіть у цьому тідлері поле finish_button значення True - це можна зробити, використовуючи $:/plugins/inmysocks/WizardWizard/Wizard Setup. В результаті на останньому слайді замість кнопки Forward (Вперед) буде відображатися кнопка завершення слайд-шоу.

За бажання ви можете модифікувати відображення цієї кнопки.

Модифікація стилів відображення

У файлі CSS стілів $:/plugins/inmysocks/WizardWizard/stylesheets.tid ви можете задати власні стилі відображення презентації, у відповідності до наданих там коментарів.
