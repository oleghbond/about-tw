Експорт документу із зображеннями

12-3-2017
Створення документів

Простий спосіб експорту ТВ до документів з подальшою розбивкою на фіксованого розміру сторінки здійснюється шляхом експортування у статичні HTML файли.

Якщо у ТВ використовуються внутрішні зображення (тобто тідлери типу im,age/jpeg, image/pngтощо), то вони стандартно зберігаються у статичному HTML як внутрішні об'єкти типу base64.

На жаль Word не розпізнає такі зображення після імпорту статичного HTML із вбудованими зображеннями типу base64.

Водночас це не є проблемою для ЛібреОфіс. Але при цьому збивається пропорції сторін зображення (aspect ratio) - зображення стають розтягнутими або сплющеними.

Зображення можна підкоригувати, змінивши вручну параметри його відображення вже безпосередньо у редакторі Write з пакету ЛібреОфіс.

Для цього подвійно клацніть по зображенню. З'явиться такий інтерфейс:

Далі:

  1. встановіть галочку Пропорционально
  2. натсніть кнопку Исходный размер
  3. встановіть потрібне вам значення у Ширина
  4. у полі Высота видаліть будь-яке значення і зніміть під ним галочку Относительно
  5. натисніть ОК
