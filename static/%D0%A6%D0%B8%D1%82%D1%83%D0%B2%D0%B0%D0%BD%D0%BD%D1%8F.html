Цитування

12-12-2016
Синтаксис ТВ

Є два способи створення блочних лапок в TiddlyWiki5, один з них - для багаторядкових блоків і другий - для застосування в одному рядку.

Багаторядковий текст

Синтаксис лапок для багаторядкових текстових блоків є:

<<<
Це цитований багаторядковий текст,
написаний українською мовою
<<<

Що відображається у вигляді:

Це цитований багаторядковий текст, написаний українською мовою

Блочні лапки можуть бути представлені в режимі цитування, як-то:

<<<
Комп'ютери як велосипед для нашого розуму
<<< Стів Джобс

Що відображається у вигляді:

Комп'ютери як велосипед для нашого розуму

Стів Джобс

CSS класи можуть бути додані до блоку цитати:

<<<.myClass.another-class
Операційні системи, як цегляна стіна для наших умов
<<< Ніхто

Що відображається у вигляді:

Операційні системи, як цегляна стіна для наших умов

Ніхто

Відповідний HTML-виглядає так:

<blockquote class="tc-quote myClass another-class"><p>Операційні системи, як цегляна стіна для наших умов
</p><cite>Ніхто</cite></blockquote>

Однорядковий текст

Синтаксис однорядкового цитування насправді є розширенням синтаксису списків у вікі-тексті. Наприклад:

> Цитований текст
> Ще один рядок цитованого тексту

Що відображається у вигляді:

Цитований текст

Ще один рядок цитованого тексту

Ви можете також вкладати цитати одна в одну, як-то:

> Цитата першого рівня
>> Вкладена цитата (другого) рівня
> Ще одна цитата першого рівня

що відображається таким чином:

Цитата першого рівня

Вкладена цитата (другого) рівня

Ще одна цитата першого рівня

Можна також змішати цитати з елементами списку іншого стилю. Наприклад:

* Перший елемент списку
** Другий елемент списку
**> Цитата
**> Інша цитата
* Третій елемент списку

Що відображається у вигляді:

  • Перший елемент списку
    • Другий елемент списку

      Цитата

      Інша цитата

  • Третій елемент списку
